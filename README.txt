QueueGames

QueueGames is a collection of lightweight, second-monitor-friendly games designed to be played during downtime — queues, loading screens, or while multitasking.


Vision
Create simple, satisfying games that:
Require low attention
Offer meaningful progression
Feel rewarding in short bursts

Philosophy
Small scope, finished projects
Learn by building
Iterate quickly
Focus on feel over complexity

Projects
(coming soon) Bubble Popper
(future) Idle / progression-based games

Ideas
See /ideas for early concepts and experiments.

Status
Early development — building the first set of games and defining the brand.
